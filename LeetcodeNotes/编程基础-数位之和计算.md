# 数位之和计算

设一数字 $x$​ ，向下取整除法符号 $//$​ ，求余符号 $\odot$​，则有：

- $x \odot 10$​ ：得到 $x$​ 的个位数字；
- $x // 10$​ ： 令 $x$​ 的十进制数向右移动一位，即删除个位数字。

因此，可通过循环求得数位和 $s$​​ ，数位和计算的`封装函数`如下所示：

```C++
int sums(int x){
    int s = 0;
    while(x != 0) {
        s += x % 10;
        x = x / 10;
    }
    return s;
}
```

`注` 思路为 **除十取余**
